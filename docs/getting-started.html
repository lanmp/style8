包含的内容

Style8 提供了两种形式的压缩包，在下载下来的压缩包内可以看到以下目录和文件，这些文件按照类别放到了不同的目录内，并且提供了压缩与未压缩两种版本。

预编译版

下载压缩包之后，将其解压缩到任意目录即可看到以下（压缩版的）目录结构：

style8/
├── css/
│   ├── style8.css
│   ├── style8.css.map
│   └── style8.min.css
├── js/
│   ├── style8.js
│   └── style8.min.js
└── fonts/
    ├── Linearicons-Free.eot
    ├── Linearicons-Free.svg
    ├── Linearicons-Free.ttf
    ├── Linearicons-Free.woff
    └── Linearicons-Free.woff2

上面展示的就是 Style8 的基本文件结构：预编译文件可以直接使用到任何 web 项目中。我们提供了编译好的 CSS 和 JS (style8.*) 文件，还有经过压缩的 CSS 和 JS (style8.min.*) 文件。同时还提供了 CSS 源码映射表 (style8.*.map) ，可以在某些浏览器的开发工具中使用。同时还包含了来自 Linearicons-Free 的图标字体。

Style8 源码

Style8 源码包含了预先编译的 CSS、JavaScript 和图标字体文件，并且还有 LESS、JavaScript 和文档的源码。具体来说，主要文件组织结构如下：

style8/
├── less/
├── js/
├── fonts/
└── dist/
    ├── css/
    ├── js/
    └── fonts/

less/、js/ 和 fonts/ 目录分别包含了 CSS、JS 和字体图标的源码。dist/ 目录包含了上面所说的预编译 Style8 包内的所有文件。docs/ 包含了所有文档的源码文件，examples/ 目录是 Style8 官方提供的实例工程。除了这些，其他文件还包含 Style8 安装包的定义文件、许可证文件和编译脚本等。

基本模板

使用以下给出的这份超级简单的 HTML 模版。我们强烈建议你对这些实例按照自己的需求进行修改，而不要简单的复制、粘贴。

拷贝并粘贴下面给出的 HTML 代码，这就是一个最简单的 Style8 页面了。

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- 上述3个meta标签*必须*放在最前面，任何其他内容都*必须*跟随其后！ -->
    <title>Style8 Template</title>

    <!-- Style8 -->
    <link href="css/style8.min.css" rel="stylesheet">

    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
      <script src="js/html5shiv/3.7.2/html5shiv.min.js"></script>
      <script src="js/respond.js/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>你好，世界！</h1>

    <!-- jQuery (necessary for Style8's JavaScript plugins) -->
    <script src="js/jquery/1.11.2/jquery.min.js"></script>
    <!-- Include all compiled plugins (below), or include individual files as needed -->
    <script src="js/style8.min.js"></script>
  </body>
</html>
